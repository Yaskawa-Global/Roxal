1. Downlaod "VxWorks SDK for Raspberry Pi 4" from https://labs.windriver.com/downloads/wrsdk_prev.html and install it.
2. Go to Roxal project source folder.
3. Run build_antlr_vxworks.sh, build_ffi_vxworks.sh, build_icu_vxworks.sh to build external libraries using vxworks tool chain.
4. Run build_roxal_vxworks.sh 
5. Cheeck the build_vxworks folder if roxal.vxe generated
